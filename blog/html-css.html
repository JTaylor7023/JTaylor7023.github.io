    • Introduction

    So you want to learn the difference between Id's and Classes in simple laymens term?

    Well you can follow the links in this blog post to learn more.

    What you need to know

    Id's and Classes are labels or hooks that we attach to elements to style or target them

    • Id: It's like a name tag and is unique.
    • Classes: Is like a group badge and can be shared.

    Right, lets get a bit more specific:

    Id's

    Id's are unique. That means you can only use them on ONE elemnt. This may not appear important but if you use an Id on more than one element your code will not pass.

    • Each element can only have one Id.
      Each page can have only one element with that ID
    • Classes

    Classes are not unique.

    • You can use the same class on multiple elements.
      You can use multiple classes on the same element.
    • Things to know
    • CSS doesn't care.

    In CSS there is nothing you can't do with a class and vice versa

      Javacript cares.

    Javascript depends on there being one element with a particular Id

      If you don't need them don't use them.
