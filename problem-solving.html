4. Generating a Random Monster

To simulate randomness, I used the Math.random() and Math.floor() methods. This is a common technique for choosing a random item from an array:

function getRandomMonster() {
    const randomIndex = Math.floor(Math.random() * monsters.length);
    return monsters[randomIndex];
  }
